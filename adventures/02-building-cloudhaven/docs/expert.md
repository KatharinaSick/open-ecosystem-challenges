Unlike the beginner and intermediate levels, the workflow files for this challenge are located in `.github/workflows/` 
(not in the challenge directory). This is because GitHub Actions requires workflow files to be in that location.

When editing workflows, navigate to:

- `.github/workflows/adventure02-expert-validate-and-plan.yaml`
- `.github/workflows/adventure02-expert-drift-detection.yaml`
- `.github/workflows/adventure02-expert-apply.yaml`
